var#

Compute the variance of a tensor. ddof can be used optionally to control the bias term in the denominator

template<typename OutType, typename InType>
void __MATX_INLINE__ matx::var(OutType dest, const InType &in, int ddof = 1, int stream = 0)#

Compute a variance reduction

Computes the variance of the input according to the output tensor rank and size

Template Parameters:

  • OutType – Output data type

  • InType – Input data type

Parameters:

  • dest – Destination view of reduction

  • in – Input data to reduce

  • ddof – Delta Degrees Of Freedom used in the divisor of the result as N - ddof. Defaults to 1 to give an unbiased estimate

  • stream – CUDA stream ID

template<typename OutType, typename InType, typename Executor>
void __MATX_INLINE__ matx::var(OutType dest, const InType &in, Executor &&exec, int ddof = 1)#

Compute a variance reduction

Computes the variance of the input according to the output tensor rank and size

Template Parameters:

  • OutType – Output data type

  • InType – Input data type

  • Executor – Executor type

Parameters:

  • dest – Destination view of reduction

  • in – Input data to reduce

  • exec – Executor type

  • ddof – Delta Degrees Of Freedom used in the divisor of the result as N - ddof. Defaults to 1 to give an unbiased estimate

template<typename OutType, typename InType, int D, typename Executor>
void __MATX_INLINE__ matx::var(OutType dest, const InType &in, const int (&dims)[D], Executor &&exec, int ddof = 1)#

Compute a variance reduction

Computes the variance of the input according to the output tensor rank and size

Template Parameters:

  • OutType – Output data type

  • InType – Input data type

  • D – Num of dimensions to reduce over

  • Executor – Executor type

Parameters:

  • dest – Destination view of reduction

  • in – Input data to reduce

  • dims – Array containing dimensions to reduce over

  • exec – Executor to use for reduction

  • ddof – Delta Degrees Of Freedom used in the divisor of the result as N - ddof. Defaults to 1 to give an unbiased estimate

Examples#

var(t0, t1, exec);
